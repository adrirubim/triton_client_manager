# Claude Memory

This folder stores context across sessions for the triton_client_manager/MANAGER project.

## Files
- `session_log.md` — chronological log of changes and decisions
- `plans.md` — current and future plans
- `notes.md` — miscellaneous notes, gotchas, and needed things

## How to use
At the start of each session, read these files to restore context.
When the user asks "do you remember..." or "what did we do...", consult these files.
