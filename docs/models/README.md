# docs/models

This directory is reserved for model repository documentation, `config.pbtxt` templates, and deployment guides.

### 🚫 Critical Safety Rules:
- **NO BINARIES:** Do not upload weights or large artifacts here (`.onnx`, `.pt`, `.safetensors`, `.bin`, `.engine`).
- **Storage:** Use external storage (MinIO/S3) or CI artifacts for heavy model files.
- **Content:** Only Markdown guides, JSON/Protobuf snippets, and folder structure examples are allowed.
Utilitza el codi amb precaució.
